* 当前设计规范，只使用了 `primary` 和 `default`。

### 迁移说明

sdistro-usercenter-f2e 中，使用了自定义的 ```<SdButton>``` 组件。对标参数：

| 参数 | SdButton | ElButton |
| ----- | ----- | ------ |
| type | `primary` | `primary` |
| type | `reset` | `default` |
| type | `text` | 使用 `<ElButton text />` 方式 |
| size | normal | large |
| size | medium | default |
